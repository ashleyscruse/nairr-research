# Formatting and Submission

## Anonymization (Double-Blind Review)

Many venues require it. If so:
- Remove your name and affiliation
- Replace "our previous work (Smith, 2024)" with "prior work (Anonymous, 2024)"
- Remove identifying acknowledgments
- Remove file metadata (File > Properties)
- Do not link to your GitHub repo (share after acceptance)

## Reference Management

### Zotero to LaTeX/Overleaf
1. Select papers in Zotero > Right-click > Export > BibTeX
2. Upload `references.bib` to your project

### Zotero to Word
1. Install Zotero Word plugin
2. Use Zotero toolbar to insert citations
3. Select citation style (APA, IEEE, ACM)

### Verify Every Citation
- Does the paper exist? (Google Scholar)
- Does it say what you claim? (Read the abstract)
- Is the citation info correct? (Authors, year, title)
- AI-generated citations must be verified

## Submission Costs

| Fee Type | When | Range |
|---|---|---|
| Submission fee | When you submit | $0-100 |
| Registration | If accepted (conferences) | $200-800 |
| Page charges | Over page limit | $100-200/page |
| Open access | If you want OA (journals) | $500-3000 |

Ask your institution about publication support funds.

## Submission Checklist

- [ ] Paper formatted to venue specs
- [ ] All figures at required resolution
- [ ] All citations verified
- [ ] Anonymized (if double-blind)
- [ ] File metadata cleaned
- [ ] Supplementary materials prepared
- [ ] Co-author approvals obtained
- [ ] Submission fee paid
- [ ] Confirmation email received

## After This Week

Your paper may not be submission-ready on Day 5. That is normal. What you should have:
- A complete first draft
- All figures and tables
- A target venue
- A Zotero bibliography
- A revision timeline

The revision process typically takes 2-4 weeks. Plan for it.
